# BidBlitz
test-5-assignment

# How to run
In the first /src directory run command:
```bash
npm install
npm run dev
```

# How to run unit tests
Also in the first /src directory run command:
```bash
npm run test
```

# Architecture
I decided to split the components into views that use different routes.
I didn't want to use alot of libraries so the only ones I used (not including vite's) are:
* vitest - for unit tests.
* react-router - for easy route management.
